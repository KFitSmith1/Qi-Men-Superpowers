快速开始Quick start

本页为项目主页；完整应用需在本地运行（前端调用 Node.js API，API 包装纯 Bash 奇门引擎）。
This page is the project home; the full app runs locally — the UI talks to a Node.js API wrapping a pure-Bash Qi Men engine. Requires Node.js ≥ 18 and Bash; zero npm dependencies.

git clone https://github.com/KFitSmith1/Qi-Men-Superpowers.git
cd Qi-Men-Superpowers
node server/src/index.js
# → http://localhost:8787

# or with Docker:
docker compose up --build

Engine: qmenpowers (GPL-3.0, pure Bash, 置闰法 · 时家转盘奇门) · optional Astrology-API.io precision mode
